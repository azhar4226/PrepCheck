Method 2: Browser Extension

Install the custom extension:

  1. Open Chrome/Edge and go to chrome://extensions/
  2. Enable "Developer mode"
  3. Click "Load unpacked"
  4. Select the folder: launchdarkly-blocker-extension/

Method 3: System-wide Hosts File

Run the blocking script:

cd /Users/apple/Desktop/PrepCheck && sudo ./block-launchdarkly.sh

Method 4: VS Code Settings

Disable telemetry in VS Code:

  1. Open VS Code settings (Cmd+,)
  2. Search for "telemetry"
  3. Set "Telemetry: Telemetry Level" to "off"
  4. Restart VS Code
