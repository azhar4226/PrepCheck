üîç What Was Fixed
Problem:
  • Frontend was sending full option text as answers
  • Backend was expecting option letters (A, B, C, D)
  • This caused all answers to be marked as incorrect
Solution:
  • Modified frontend to send option letters instead of text
  • Updated option structure to include both letter and text
  • Radio button values now use the letter (A, B, C, D)
