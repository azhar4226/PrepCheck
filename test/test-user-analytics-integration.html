⚡ API Test Commands

1. Login as Admin:

curl -X POST http://localhost:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email": "admin@prepcheck.com", "password": "admin123"}'

2. Test User Analytics (replace token):

curl -X GET "http://localhost:8000/api/quiz/user-analytics?days=30" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json"

3. Test Admin User Analytics (replace token and user_id):

curl -X GET "http://localhost:8000/api/admin/user/4/analytics?days=7" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json"
